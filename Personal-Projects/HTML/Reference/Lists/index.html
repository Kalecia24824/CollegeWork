Types of Lists

by Kalecia McNeal

Lists are very important because they organize information into one place. There are other elements and content that can organize information. But for now, let's view the different types which are unordered, ordered and description.


Unordered Lists

unordered lists are lists that are not numbered. These lists will have a default of a black circle unless you use CSS to change the style of the element. But for now, here is an example of an unordered list:

Grocery List

  • Milk
  • Cheese
  • Bread

As you can see this list has three items someone needs for groceries


Ordered Lists

Up next is ordered lists. These lists are important because they have an order. Ordered lists can also be used to categorize items. An example of this can be see below:

Colors of the Rainbow:

  1. Red
  2. Orange
  3. Yellow
  4. Green
  5. Blue
  6. Indigo
  7. Violet

These colors are in order based on number and hierarchy. Lists like these are crucial when putting items together in order.


Description Lists

Lastly, but not least, we have description lists which are used to add descriptions or summaries to list items. Like this:

Fruits

Apples
A shiny red fruit
Strawberries
A small red berry with seeds on the outside
Banana
A curved yellow fruit

As you can see, the description list contains the (dl) tag which is the list itself, (dt) the term and (dd) which is the description describing each item


Finally, you learned about all three types of lists from unordered to description lists. I hope this guide helps you in deciding which list you need for the job!
