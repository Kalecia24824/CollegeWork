Today, we will learn about and how to use images in HTML. Now there are various of ways to do it but we will take a look at three main ways.


The Regular Way

Before revealing the first image, we need to learn how the img element works. The image tag or img is used to embed images in a web page. These tags allow the user to visualize content online.

Now, the img tag has two attributes: src which specific the path and alt which provides alternative text for accessibility. Now, that you understand the terms here is what the syntax of the img element would look like: (img src="url" alt="text")
