All about Tables

Today we will learn about tables. Now, tables are very useful for recording and displaying information just like lists except they are contained into boxes.

Regular tables have rows and columns with the first cells always serving as headers. In HTML, tables have the table element, table headers, table rows and cells. Now there are more elements involved with tables but for now just focus on the four above.


Table Cells

Now, the first example below is a regular table focusing on the cells which is the (td) element. TD stands for table data as you can see here:

Adrianna Kalecia Aurora

As you can see there is no headers which just places the data in a row. So the user has no idea what this information can be about.


Table Rows

Now for our second example, it will have rows which is much more helpful:

Adrianna Kalecia Aurora

See this table is much easier but it is still missing the headers which provide crucial information
