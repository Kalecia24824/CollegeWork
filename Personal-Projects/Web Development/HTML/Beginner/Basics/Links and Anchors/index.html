Links

Today, we will learn about links. Links are important because they can help connect the user to different info. Before we view some examples, you need to understand how they work

A link consist of three things: the a tags, href and target attributes. The a tags stand for anchor, the href is the hyperlink url and the target attribute determines where the document will be opened at.
