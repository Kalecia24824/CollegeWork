HTML Semantics

Today for our final installment in Basics, we will learn about semantics and how they work

List of Semantic Elements

Since there are so many elements, I decided it would be best to place them in a list. Now there will be at least two terms that are not semantic because they only apply to HTML. With that out of the way, let's get started!

Article
Defines independent, self-contained content

Aside
Defines content aside from the page content

Details
Defines additional details that the user can view or hide

Div
Not a sematic element. Designed to help keep information contained into groups in HTML.

Figcaption
Defines a caption for a (figure) element

Figure
Specifies self-contained content, like illustrations, diagrams, photos, code listings, etc.

Footer
Defines a footer for a document or section

Header
Specifies a header for a document or section

Main
Specifies the main content of a document

Mark
Defines marked/highlighted text

Nav
Defines navigation links

Section
Defines a section in a document

Span
Marks up parts of text or a document in HTML. Also not a semantic element

Section
Defines a section in a document

Summary
Defines a visible heading for a (details) element

Time
Defines a date/time

And that is all of the elements used in HTML semantics! I hope this guide helps you with learning about Semantics.
